# Используемые символы

## Описание

Для достижения лаконичности изложения информации необходимо использовать специальные математические обозначения. Этот раздел должен служить **быстрым справочником** для понимания логики конспектов.

## Логические связки и утверждения

Эти символы используются для построения формальных утверждений и выражений.

| Символ | Значение (Описание) | Пример Использования |
| :---: | :---: | :---: |
| $\implies$ | **Импликация/cледствие** (*если $P$, то $Q$*) | $a=2 \implies a^2=4$ |
| $\iff$ | **Эквивалентность** (*$P$ только тогда, когда $Q$*) | $x+5=y-2 \iff y=x+7$ |
| $\neg$ | **Отрицание** (*не $P$*) | $P=0 \implies \neg P=1$ |
| $\lor$ | **Дизъюнкция** (*$P$ или $Q$*) | $P=0,\ Q=1 \implies P\lor Q=1$ |
| $\land$ | **Конъюнкция** (*$P$ и $Q$*) | $P=1,\ Q=0 \implies P\land Q=0$ |
| $\forall$ | **Квантор всеобщности** (*для всех*, *для любого*) | $\forall x\in\mathbb{N},\ x>0$ |
| $\exists$ | **Квантор существования** (*существует*) | $\exists y \in \mathbb{Z}: y<0$ |
| $\exists !$ | **Квантор единственности** (*существует единственный*) | $\exists!x \in \mathbb{R}: x=-a$ |
| $x:P(x)$ | **Условие** (*для $x$ истинно $P(x)$*) | $\exists x: x^2=4$ |

## Символы теории множеств

Символы, описывающие построение множеств, принадлежность элементов и отношения между множествами.

| Символ | Значение (Описание) | Пример Использования |
| :---: | :---: | :---: |
| $\in$ | **Принадлежит** (*является элементом*) | $x\in\mathbb{N}$ |
| $\notin$ | **Не принадлежит** (*не является элементом*) | $y\notin\mathbb{Z}$ |
| $\left\lbrace x \mid P(x) \right\rbrace$ | **Множество, заданное условием** (*множество $x$, удовлетворяющих $P(x)$*) | $\left\lbrace x \mid x=2n+1,\ n\in\mathbb{N} \right\rbrace$ |
| $\subseteq$ | **Подмножество** (*$A$ содержится в $B$*) | $\mathbb{Z} \subseteq \mathbb{Q}$ |
| $\subset$ | **Строгое подмножество** (*$A$ содержится в $B$, но $A \ne B$*) | $\mathbb{N} \subset \mathbb{Z}$ |
| $\emptyset$ | **Пустое множество** | $\left\lbrace x \mid x\in\mathbb{R},\ x^2=-1 \right\rbrace = \emptyset$ |
| $\cap$ | **Пересечение множеств** | $A \cap B = \left\lbrace x \mid x \in A \land x \in B \right\rbrace$ |
| $\cup$ | **Объединение множеств** | $A \cup B = \left\lbrace x \mid x \in A \lor x \in B \right\rbrace$ |
| $A \setminus B$ | **Разность множеств** | $A \setminus B = \left\lbrace x \mid x \in A \land x \notin B \right\rbrace$ |

## Стандартные числовые множества

Общепринятые обозначения для наиболее часто используемых множеств чисел.

| Символ | Значение (Описание) | Определение |
| :---: | :---: | :---: |
| $\mathbb{N}$ | **Натуральные числа** | $\left\lbrace 0, 1, 2, \dots \right\rbrace$ |
| $\mathbb{Z}$ | **Целые числа** | $\left\lbrace \dots, -2, -1, 0, 1, 2, \dots \right\rbrace$ |
| $\mathbb{Q}$ | **Рациональные числа** | $$\left\lbrace \frac{p}{q} \mid p,q\in\mathbb{Z},\ q\neq 0 \right\rbrace$$ |
| $\mathbb{R}$ | **Действительные (вещественные) числа** | Множество всех рациональных и иррациональных чисел. |
| $\mathbb{C}$ | **Комплексные числа** | Множество чисел $a+bi$, где $a, b \in \mathbb{R}$ и $i^2=-1$. |
